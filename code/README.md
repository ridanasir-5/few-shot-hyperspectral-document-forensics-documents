# Code

This directory contains the implementation of the proposed few-shot hyperspectral forensic document analysis framework.

The source files include the unified forensic pipelines and ablation study scripts used to generate the experimental results reported in the associated manuscript.


### Repository Status

The repository currently includes the experimental pipelines, pretrained checkpoints, and evaluation outputs used in this study. Certain implementation components (e.g., `model.py`) are being prepared for public release and will be made available following the completion of the manuscript review process.
